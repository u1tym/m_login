Vue + Nginx + FastAPI で構成されたシステムを作っている。


要件：
- 認証専用の FastAPI を1つ作りたい
- ログイン成功時に JWT を発行し、HttpOnly Cookie にセットする
- Cookie の domain は指定しない（IP アクセスのため）
- Cookie の属性は HttpOnly, Secure, SameSite=Lax, Path="/"
- JWT の有効期限は 30 分
- SECRET_KEY と ALGORITHM は環境変数から読み込む
- /login と /logout を実装する
- /me で Cookie の JWT を検証してユーザ情報を返す
- パスワードはハッシュ化して検証する
- CORS 設定は Vue からのアクセスを許可し、credentials を許可する

なお、
Nginx をリバースプロキシとして使っている。
Vue は Nginx で配信し、FastAPI は /api/*** にプロキシしている。
この認証は、/api/auth
他には、/api/recipe、/api/schedule などがある。

FastAPI は複数あるが、認証は1つの FastAPI (/api/auth) だけが担当する。
他の FastAPI は Cookie の JWT を検証するだけ。

アクセスは IP アドレス（IPv4/IPv6）で行われるため、Cookie の domain は指定しない。

DBの接続先は、.envで作成して。
初期値は、
Server: localhost
Database: tamtdb
Port: 5432
Username: tamtuser
Password: TAMTTAMT
